# Karpor 着陆页

[English](./README.md) | [中文](./README_zh.md)

### 简介

这是 Karpor 的官方着陆页。Karpor 是一个专注于搜索、洞察和 AI 能力的 Kubernetes 资源管理器。使用 Next.js 和 Tailwind CSS 构建，提供现代化和响应式的用户界面，展示 Karpor 的特性。

### 特性

- 🔍 **灵活搜索**：跨集群的实时同步和智能查询
- 📊 **资源洞察**：全面的监控和交互式拓扑视图
- 🤖 **AI 助手**：基于 LLM 的分析和专家级诊断

### 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm start
```

### 技术栈

- Next.js 14
- React 18
- Tailwind CSS
- TypeScript
